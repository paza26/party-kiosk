# Pre-Task Hook

Before executing ANY task, check:

1. **Is this a Git operation?**
   → YES: MUST use git-workflow-manager

2. **Is this documentation work?**
   → If >10 min: MUST use code-documentation-specialist

3. **Is this React Native code?**
   → YES: MUST use react-native-architect

4. **Is this testing?**
   → YES: MUST use react-native-test-engineer

5. **Is this UX evaluation?**
   → YES: MUST use ui-ux-senior-analyst

See `.claude/agent-automation-rules.md` for complete rules.
